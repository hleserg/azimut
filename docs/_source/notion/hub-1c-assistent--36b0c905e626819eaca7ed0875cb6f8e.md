> Хаб проекта **Агент-консультант по 1С ERP** (репозиторий [github.com/hleserg/1c-consultant](http://github.com/hleserg/1c-consultant)).
> Локальный MCP-агент — компетентный консультант по типовой конфигурации 1С ERP. Пользователи: Сергей и мама.
> Задачи ведутся в Linear (команда Hleserg, проект «Агент-консультант по 1С ERP»). Этот хаб — для рабочих материалов вне Linear-эпика.
## Документы проекта
Ниже — три ключевых рабочих файла. Поддерживаются вручную, синхронизированы с гитом.
- **Решения (decisions)** — принятые архитектурные и поведенческие решения по борьбе с галлюцинациями и поиску.
- **Операционное руководство (runbook)** — для руководителя: мониторинг, регламенты, калибровка, эксплуатация.
- **Открытые вопросы (questions)** — сводный список нерешённого. Приоритет №1 — обработка кода.
## Контекст в двух словах
Агент не «улучшенный поиск», а спец: собирает контекст проблемы, проводит ресёрч по источникам, отвечает ТОЛЬКО из найденного без фантазий, сам определяет нехватку данных и идёт искать. Реалистичная планка — «почти всегда сам в источники, по делу», не 100%.
Источники данных (легальные): выгрузка конфигурации в XML, офлайн-версия ИС 1С:ИТС, портал техдокументации платформы.

**Стек (актуализировано 2026-05-27 по `_resolutions.md` #5):**
- **Язык/окружение:** Python ≥3.11, uv, FastMCP.
- **Ядро понимания кода 1С:** форк `bsl-atlas` под именем «Азимут» (AGPL-3.0, реш. 1.1).
- **Эмбеддинги:** BGE-M3 (локально, мультиязычный, dense+sparse).
- **Хранилище:** Qdrant embedded в локальном режиме / Qdrant server в онлайн/мульти-аренде (реш. 1.4 + развилки темы 7).
- **Реранкинг по умолчанию:** локальный BGE-reranker-v2-m3. **Cohere Rerank v4 — опциональный апгрейд по решению Сергея, грант не тратим на пользователя автоматически** (реш. 1.8).
- **Разговорная модель по умолчанию:** **DeepSeek V4** (Flash — основной, Pro — для тяжёлого разбора кода). Работает из РФ без VPN. Запас: Claude (премиум дома + eval-эталон), Qwen3.7-Max, YandexGPT/GigaChat — для комплаенса (реш. 1.8a).
- **MCP-клиент по умолчанию:** **Cherry Studio** (бесплатный опенсорс, AGPL-3.0 + dual licensing, DeepSeek + полный MCP). Мама — только чат; Сергей — Cherry+DeepSeek everyday + Claude Desktop дома для премиум-задач; mini-ai-1c — Сергею для подачи кода из Конфигуратора (реш. 1.7a).
- **Дополнительные MCP рядом:** `alkoleft/mcp-bsl-platform-context` (справочник API платформы, MIT, реш. 1.6).

📍 **Актуальный источник истины по архитектуре — `design-system-v2.md`** (живой документ). Этот хаб — вводная.
