## Если что-то не работает

* **BGE не загружается** → проверь интернет и место на диске (\~1.5GB для модели). Путь к кешу в конфиге
* **Поиск возвращает пустой результат на реальный вопрос** → проверь что индексация прошла успешно (КТ3), и что коллекция в Qdrant не пустая
* **Реранкер падает** → если нет ключа Cohere — должен работать локальный fallback BGE-reranker. Проверь в конфиге `reranker.provider`
* **Всё медленно (> 30 сек)** → скорее всего BGE грузится на каждый запрос. Проверь что синглтон работает
