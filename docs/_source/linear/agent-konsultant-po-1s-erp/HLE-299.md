Единый поисковый слой над всеми источниками. **Решает «модель читает первый док наполовину»** — дотошность обеспечивает СЕРВЕР. Спроектирован под **переключаемые стратегии** (обкатка — задача 11).

## Что сделать

* Эмбеддинг через **FlagEmbedding** (BGE-M3): dense + sparse. Хранить ОБА представления на чанк.
* **Переключаемые стратегии** через конфиг (паттерн «стратегия», добавление новой не требует переписывания слоя):
  * `dense` — только плотный (базлайн);
  * `hybrid` — dense + sparse;
  * `hybrid_rerank` — гибрид + **Cohere Rerank v4** (API);
  * фолбэк-реранк на локальный BGE-reranker без ключа Cohere.
* **Полные разделы, а не обрывки** — подтягивать весь смысловой раздел по иерархии (задача 4).
* **Авторазворачивание вложенности** — список связанных/дочерних + сигнал «есть ещё N разделов, вот id».
* **Сигнал полноты** — сколько совпадений, что выдано, есть ли непросмотренное.
* Поиск по структуре конфы (задача 3); кросс-источниковый поиск (конфа+дока) с пометкой источника.

## Definition of done

* Все стратегии работают, переключаются одной настройкой без правки кода слоя.
* `hybrid_rerank` без ключа Cohere → автофолбэк на локальный reranker, без падения.
* Вложенная тема → корневой раздел целиком + перечень дочерних.
* В ответе метаданные полноты.
* Несколько представлений на чанк в индексе (готовность к shadow-эксперименту).
