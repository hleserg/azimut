## Как проверить структуру репо

Структура должна быть такой (примерно):

```
1c-consultant/
  src/
    config.py
    registry.py
    ...
  tests/
  data/          ← в .gitignore
  .env.example
  AGENTS.md
  pyproject.toml
```

## Если что-то не работает

* `uv sync` **падает** → читай `pyproject.toml`, возможно конфликт версий. Попробуй `uv sync --reinstall`
* **Конфиг не читается** → проверь что `.env` создан из `.env.example` и заполнен путями
* **Registry падает с ошибкой SQLite** → убедись что папка `data/` существует и доступна на запись
* **Тесты падают** → читай traceback, скорее всего путь к тестовым данным не настроен
