## Дизайн

* **Основной судья — Claude.** ВНЕ retrieval-конвейера (не эмбеддит, не реранжирует) → независимый арбитр. Сильнее в рассуждении по сложной доменной выдаче.
* **Cohere Command — опц. второй голос.** НЕ основной: Cohere участвует в стратегиях (rerank/мультимодал) → конфликт интереса. Но как второй независимый голос интересен — расхождение Claude vs Command = сигнал.
* Судья работает ОФЛАЙН по накопленным логам, пакетно.

## Что делает

* Вход: запрос + выдачи стратегий, **обезличенные** (A/B/C, чтобы судья не знал какая чья — снижает предвзятость).
* Выход: структурированный вердикт — какая полнее/релевантнее, почему, оценка по шкале.
* Агрегация по многим запросам → какая стратегия чаще выигрывает.

## Definition of done

* Судья на Claude прогоняет пакет логов, выдаёт структурированные вердикты.
* Выдачи подаются обезличенно.
* Command опц. как второй голос; расхождение судей — в отчёт.
* Результаты в формат для анализа (задача 13).
