## Что сделать

* Скрипт/ноутбук: свести логи. По каждой стратегии — доля побед (по фидбэку / по неявным сигналам / по LLM-судье отдельно), средние тайминги, стоимость (Cohere $).
* Сводный отчёт: «стратегия X выигрывает N% по фидбэку, M% по судье, стоит $K/1000, медленнее на T мс» — качество vs цена vs скорость в одной таблице.
* Подсветка расхождений между источниками оценки.
* Рекомендация: что закрепить в production, что выкинуть, что докрутить.

## Definition of done

* Сводный отчёт по стратегиям (победы/тайминги/стоимость) генерируется на логах.
* Видно, какую стратегию выбрать в production, и обоснование.
* Обновлён ADR (задача 2): отложенные решения закрыты зафиксированным выбором со ссылкой на данные.
