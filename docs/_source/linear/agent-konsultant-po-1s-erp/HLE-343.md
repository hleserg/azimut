## Sentry

После индексации загляни в Sentry — там должны появиться трейсы индексации. Если видишь ошибки — читай, разбирайся или открывай issue.

## Если что-то не работает

* **Qdrant не стартует** → проверь что embedded-режим инициализируется в коде, не нужен отдельный процесс
* **OCR не работает** → `tesseract --version` в терминале. Если не установлен — `brew install tesseract` (Mac) или `apt install tesseract-ocr tesseract-ocr-rus` (Linux/WSL)
* **LibreOffice не конвертирует** `.doc` → `libreoffice --version`. Если нет — установи LibreOffice
* **PII не маскируется** → `uv run python -c "import spacy; spacy.load('ru_core_news_sm')"` — если ошибка, модель не скачана: `uv run python -m spacy download ru_core_news_sm`
* **Реестр не обновляется** → проверь права на папку `data/`
