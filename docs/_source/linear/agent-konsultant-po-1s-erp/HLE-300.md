## Поисковые инструменты

* `search_documentation` — гибридный поиск по доке. ОБЯЗЫВАЕТ: «Вызывай перед любым ответом про конфигурацию/платформу. Отвечать из памяти про 1С ЗАПРЕЩЕНО — только из результатов.»
* `get_section`, `search_metadata`, `get_object`, `list_related`, `show_image` (см. подробности по именам).

## Условный показ Cohere

* Инструменты/стратегии на Cohere (rerank v4, мультимодал, Command-судья) **видны/предлагаются только при наличии ключей Cohere**. Нет ключа — опция не фигурирует, никаких падений.

## Принципы описаний

* Описание формулирует КОГДА модель ОБЯЗАНА вызвать инструмент. Запрет отвечать из памяти по 1С. Инструменты возвращают сигнал полноты.

## Definition of done

* Сервер поднимается, подключается к десктопу, инструменты вызываются.
* На вопрос по 1С без явной команды Claude сам идёт в `search_documentation`.
* Без ключей Cohere соответствующие опции не видны; всё работает на локальном стеке.
* README: подключение к десктопу (конфиг MCP, ручной путь).
