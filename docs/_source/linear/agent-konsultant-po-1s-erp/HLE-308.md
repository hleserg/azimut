Адаптеры к API Cohere как переключаемые провайдеры. Грант $1000 — на то, где Cohere первоклассен: реранкинг и (на обкатку) мультимодальный эмбеддинг картинок.

## Адаптеры

* **Rerank v4** (`rerank-v4.0-pro`/`fast`) — мультиязычный, 100+ языков, длинные документы/таблицы/код. В стратегию `hybrid_rerank` (задача 5). \~$2/1000 поисков.
* **Embed 4** (`embed-v4.0`) — мультимодальный, эмбеддит скриншоты напрямую (без OCR). Кандидат для пути «картинки» на обкатку (задачи 11-12). Матрёшка 256/512/1024/1536, квантизация.
* **Command** — опционально второй голос LLM-судьи (задача 12).

## Требования

* Единый интерфейс провайдера: BGE (локально) и Cohere (API) взаимозаменяемы за абстрактным классом — переключение через конфиг.
* **Условная активация:** только при наличии ключа. Нет ключа — провайдер недоступен, опции не предлагаются (задача 6), фолбэк на локальный стек.
* Учёт стоимости: логировать потраченное по Cohere на запрос (для задачи 13).
* Обработка ошибок API (лимиты, сеть): не валить запрос, фолбэк на локальное.

## Definition of done

* Rerank v4 работает в `hybrid_rerank`; без ключа → фолбэк на BGE-reranker.
* Embed 4 доступен как опц. путь для картинок.
* Command подключается как второй судья по флагу.
* Стоимость Cohere логируется на каждый вызов.
* Сбой API не роняет пользовательский запрос.
